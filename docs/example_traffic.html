Información técnica

¿Cómo funciona?

Al pulsar el botón estás haciendo una consulta al servicio egov. Es un servicio que permite a cualquier persona seleccionar los datos públicos que necesita con un sencillo lenguaje de consultas. Por ejemplo, esta es la pinta que tiene la consulta para obtener la lista de radares de tráfico.

{
  poi {
    ... on TrafficRadar {
      roadName
      adminName2
    }
  }
} Egov recibe la consulta y va a buscar los datos solicitados a fuentes públicas, como la página de la Dirección General de Tráfico, que actualmente no los ofrecen de forma realmente abierta y accesible. Cuando tiene los datos, se los devuelve a este sitio web en un formato adecuado para poder mostrártelos. ¡Todo esto en cuestión de segundos!

El servicio egov es una API GraphQL pública y cuenta con un entorno para que puedas probar tus propias consultas. Anímate a probarlo.
